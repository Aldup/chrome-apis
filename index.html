Sublime plugin for Chrome APIs (Apps and Extensions) installation instructions:
  • Download sublime_chromeapis_plugin.tgz
  • Extract the file and move the ChromeApis dir to ~/Library/Application Support/Sublime Text 2/Packages/

Want to port Chrome APIs to your preferred IDE? Download the latest IDE-agnostic definitions file for Chrome Apps and for Chrome Extenstions.
